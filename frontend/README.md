# Frontend - Coding Assessment Platform

React frontend for the coding assessment platform.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- Monaco Editor for code editing with syntax highlighting
- Real-time timer component
- Test results visualization
- Multi-language support (Python, Java, C++, JavaScript)
- Responsive design

## Configuration

Edit `src/services/api.js` to change the backend API URL.
